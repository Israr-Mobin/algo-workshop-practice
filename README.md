# algo-workshop-practice

Solutions and short write-ups for a handful of classic LeetCode problems, solved while practicing algorithmic patterns for technical interviews.

Each folder below has a quick README (idea -> approach -> complexity -> pattern) plus a working solution skimmable in under a minute.

## Problems

| Problem                              | Optimal Solution Pattern    |
|--------------------------------------|-----------------------------|
| Contains Duplicate                   | Hash Set                    |
| Permutations II                      | Backtracking                |
| Smallest Palindromic Rearrangement I | Greedy / Frequency Counting |
| Valid Palindrome                     | Two Pointers                |
| Valid Parentheses                    | Stack                       |

## Why

Practicing pattern recognition - knowing *when* to reach for a hash set, a stack, two pointers, etc. - rather than memorizing one-off solutions.
